****Spotify_duplicate finder****
This is an offline version of a duplicate finder for spotify.
-create a text file called track_list where you write your files
-write the playlist name and on each new line write the track spotify link
-repeat this step for each playlist
you can copy all the somgs in a playlist blicking on one and doing CTRL+a, CTRL+c, and CTRL+v 
-The track spotify link which are duplicates in multiple playlists will be printed in the console with each playlist they are in
-use the link to find the track name
